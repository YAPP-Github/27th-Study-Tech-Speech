# Android AccessibilityService

## 학습 링크
https://velog.io/@victoryu/Android-AccessibilityService

### 요약
1. 접근성 서비스는 사용자가 설정에서 직접 켰을 때만 시작되며 오직 시스템만이 서비스에 바인딩할 수 있도록 전용 권한을 반드시 요청해야 합니다.

2. 설정 방식은 정적과 동적으로 구분되나 화면 콘텐츠 탐색과 같은 특정 기능은 반드시 XML 리소스를 통한 정적 방식으로만 활성화할 수 있습니다.

3. 성능 최적화를 위해 패키지 이름과 이벤트 유형을 필터링하여 수신 범위를 제한하고 타임아웃 설정을 통해 시스템 부하를 관리해야 합니다.
